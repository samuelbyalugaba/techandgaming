# **App Name**: Tech And Gaming

## Core Features:

- Game Listing & Search: Display a searchable list of games with filtering by genre and difficulty. Uses tags for improved discoverability.
- Game Detail Page: Show detailed information about each game, including title, description, trailer, screenshots, and mechanics.
- Subdomain Play Button: Provide a direct link to play the game on its respective subdomain.
- Blog/News Section: Publish markdown-based blog posts for updates and news.
- Admin CMS: A simple CMS dashboard for adding and editing games and publishing blog posts using Firestore.
- Leaderboard Widget: Display a snippet of the leaderboard for each game, pulling data from Firestore.

## Style Guidelines:

- Primary color: Deep Indigo (#663399) for a modern, tech feel.
- Background color: Very light grayish-blue (#f0f8ff).
- Accent color: Cyan (#00FFFF) to give that retro-neon glow, for highlights and calls to action.
- Body font: 'Inter', a sans-serif font providing a modern and neutral feel.
- Headline font: 'Space Grotesk' a sans-serif font for titles and short blocks of text, in order to feel more technological
- Use a set of minimalist icons with a neon-style outline for categories, tags, and actions.
- Design a responsive, mobile-first layout with clear content sections and a focus on readability.
- Incorporate subtle hover glows and smooth transitions for a modern, interactive experience.
